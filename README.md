# Build tools project

Let's play with

- Git
- Sass
- NPM Scripts
- Static Sites Generators

## Installation

- install node and npm (npm is bundled with node)
- run `npm install`
- run `npm run build` to build the project (production build)
- run `npm run dev` to run a development environment (watch)
